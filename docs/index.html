问题与挑战

在LEO卫星通信中，波束赋型决策同时面临“信道快变”和“约束强耦合”。

01

CSI过时

LEO卫星高速运动导致信道在覆盖周期内快速变化，基于延迟CSI的优化结果容易偏离真实可行域。

02

非凸约束复杂

用户SINR、卫星功率和ARIS反射功率相互耦合，联合波束赋型形成高维非凸约束优化问题。

03

传统AO难以实时

交替优化依赖准确CSI，并需要每个时隙迭代求解，在高动态场景下计算开销和时延都不友好。

04

普通DRL缺少可行性意识

惩罚项或简单投影难以稳定处理复杂约束，容易出现收益优化与约束满足之间的冲突。

提出方案

SDAN把优化问题中的可行域知识嵌入Actor训练过程。

SDAN-DRL以LEO卫星为智能体，输出卫星波束赋型和ARIS相移的联合动作。区别于仅靠奖励惩罚的DRL，SDAN通过PFP模块在预测CSI下构造近端可行动作，将“如何满足约束”的知识传递给策略网络。

训练时，复合损失同时考虑动作价值、PFP可行性距离和CSI预测误差；推理时，训练好的SDAN只需一次DNN前向传播即可给出实时联合波束赋型决策。

创新性与核心贡献

本文贡献点

  1. 01

    面向ARIS辅助LEO卫星通信，建立覆盖周期内的序贯CSR最大化问题，联合考虑卫星/ARIS波束赋型、用户QoS、卫星功率预算和ARIS功率预算。

  2. 02

    提出SDAN-DRL框架，使Actor不仅学习高收益动作，也通过可行性引导主动学习动态约束规则，缓解受限DRL中的最优性-可行性冲突。

  3. 03

    设计PFP模块，将预测CSI下的非凸约束近似转化为可求解的QCQP，为策略输出构造最近可行动作和可学习可行边界。

  4. 04

    引入LSTM CSI预测模块与复合损失函数，使PFP在更准确的动态信道估计下工作，并联合提升CSR、QoS满足率和训练鲁棒性。

  5. 05

    从训练收敛、动态测试、消融实验、参数敏感性、系统扩展和复杂度角度验证SDAN-ARIS的性能优势与部署潜力。

核心机制

SDAN的四个关键环节

01

预测CSI

利用LSTM从滑动CSI缓存中学习时间相关性，缓解信道老化带来的决策偏差。

02

生成联合动作

Policy-DNN输出卫星波束赋型矩阵和ARIS相移矩阵，直接服务下行传输。

03

构造可行引导

PFP在SINR和功率约束下寻找最接近原始动作的可行决策，提供约束学习信号。

04

复合损失训练

价值项、可行性项和预测项共同更新Actor，使策略逐步靠近高价值可行域。

1. Training and Testing Performance

训练收敛与动态测试

  • SDAN-ARIS在训练中达到最高稳态CSR，并比TD3-ARIS和DDPG-ARIS更快进入初始收敛阶段。
  • Loss曲线呈现两阶段特征：先由PFP推动Actor靠近可行域，再在可行域附近进行价值优化。
  • 测试阶段，SDAN-ARIS在不同时间步和不同覆盖周期中保持最高动态CSR，体现跨episode鲁棒性。
  • QoS满足率超过99%，明显优于基于过时CSI的AO-ARIS；AO-ARIS整体约为62%。
SDAN-ARIS平均99.0%，四个用户分别为98.3%、98.5%、100.0%等高可靠水平
TD3-ARIS平均99.1%，但缺少PFP/LSTM协同带来的CSR与训练稳定性优势
DDPG-ARIS平均93.5%，其中UT3下降到90.1%，公平性和鲁棒性不足
AO-ARIS平均62.0%，在过时CSI下明显退化

2. Ablation Experiment and Coefficients Analysis

模块贡献与超参数影响

  • 完整SDAN-ARIS达到最高CSR，约为44.9 bit/s/Hz，跨运行方差约为1.6。
  • w/o-LSTM保留PFP后，相比TD3获得35.5%的主要CSR增益，说明PFP是效率提升的关键来源。
  • w/o-PFP相对TD3提升有限，说明仅有CSI预测不足以有效解决复杂约束下的策略探索问题。
  • 预测因子δ过低或过高都会削弱CSR；约束因子β增大时QoS更保守但CSR可能下降，体现可行性与最优性的权衡。

3. System Parameters versus Performance

功率预算与反射单元规模

  • 随着总功率预算增加，各方案CSR整体上升；SDAN-ARIS的优势在高功率区域继续扩大。
  • ARIS、PRIS、noRIS呈现稳定层级关系：ARIS > PRIS > noRIS，说明主动反射可有效缓解PRIS乘性衰落。
  • ARIS功率分配机制可为卫星卸载约3 dBW功率，同时保持更高CSR。
  • 反射单元数量R增加时，RIS辅助系统CSR提升；SDAN在ARIS、PRIS、noRIS三类系统中均保持最高CSR。

4. Complexity Analysis

低在线复杂度

AO类方法需要在每个时隙基于CSI反复迭代求解，而SDAN在离线训练后，在线阶段只需一次DNN前向传播即可输出联合波束赋型动作，更适合LEO卫星短覆盖窗口中的实时决策。
